# pydbsec

DB증권 OpenAPI Python wrapper.

## Project

- Python 3.10+, dependencies: httpx, pydantic
- Build: `hatchling` (pyproject.toml)
- Tests: `pytest tests/ -v`
- Lint: `ruff check src/ tests/`
- Type check: `mypy src/pydbsec/ --ignore-missing-imports`
- Docs: `mkdocs serve` (mkdocs-material)
